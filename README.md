# Jorik van Ruiswijk — Portfolio

Personal vibe-coded portfolio site for Jorik van Ruiswijk — CTO, Designer & Developer.

## Stack

- **React 18 + TypeScript + Vite** — modern DX, fast builds
- **Tailwind CSS v4** — utility-first styling
- **Framer Motion** — scroll-triggered animations and entrance effects
- **Lucide React** — clean icon set

## Setup

```bash
cd portfolio
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## Build

```bash
npm run build
npm run preview  # preview the production build locally
```
